# หลักสูตรวิทยาศาสตรบัณฑิต สาขาวิชาวิทยาการคอมพิวเตอร์ (ปรับปรุง 2569)
## รายละเอียดเนื้อหารายวิชาบังคับเชิงลึกที่สุด (Ultra-Detailed Complete Syllabus)

เอกสารฉบับนี้รวบรวมเนื้อหาการเรียนการสอนโดยละเอียดสำหรับวิชาเฉพาะคอมพิวเตอร์ (แผนสหกิจศึกษา) โดยมีการคละจำนวนบทเรียน (7-10 บท) ตามความเหมาะสมของเนื้อหาและความต้องการของผู้ใช้ ครอบคลุมทั้งสิ้น 27 รายวิชา

---

## 🟢 ชั้นปีที่ 1: การปูพื้นฐานแนวคิดทางคอมพิวเตอร์

### 1. 09-131-101 Introduction to Computer Science (10 บท)
*   **บทที่ 1: ประวัติและวิวัฒนาการของคอมพิวเตอร์:** เครื่องจักรยุคแรก สู่ยุคควอนตัม และวิถีการประมวลผล
*   **บทที่ 2: สถาปัตยกรรมฮาร์ดแวร์และการทำงานของ CPU:** กลไก Fetch-Decode-Execute และสถาปัตยกรรม Von Neumann
*   **บทที่ 3: ระบบตัวเลขและการแทนข้อมูล:** Binary, Hex, Floating Point และสถานะทางลอจิก
*   **บทที่ 4: การแทนข้อมูลสื่อดิจิทัล:** การเข้ารหัสตัวอักษร บีบอัดข้อมูลภาพและเสียง
*   **บทที่ 5: พื้นฐานระบบปฏิบัติการและซอฟต์แวร์:** บทบาทของ Kernel, Memory Management และ Process
*   **บทที่ 6: อัลกอริทึมและการแก้ปัญหาเชิงคำนวณ:** ฝึกการเขียน Flowchart และ Pseudocode สำหรับโจทย์ปัญหา
*   **บทที่ 7: ระบบเครือข่ายและการสื่อสารข้อมูล:** มาตรฐาน OSI Model, โปรโตคอลอินเทอร์เน็ต และ IP Addressing
*   **บทที่ 8: พื้นฐานความปลอดภัยสารสนเทศ:** แนวคิดความปลอดภัยแบบ CIA Triad และการป้องกันภัยเบื้องต้น
*   **บทที่ 9: จริยธรรม สังคม และกฎหมายคอมพิวเตอร์:** PDPA, กฎหมายไซเบอร์ และการใช้งานเทคโนโลยีอย่างรับผิดชอบ
*   **บทที่ 10: เทคโนโลยีอุบัติใหม่:** ความเข้าใจเบื้องต้นเกี่ยวกับ AI, Blockchain, และ Quantum Computing

### 2. 09-131-102 Structured Programming (10 บท)
*   **บทที่ 1: ปูพื้นฐานภาษาโปรแกรมและการติดตั้งสภาพแวดล้อม:** การทำงานของ Compiler และการเตรียม Tools
*   **บทที่ 2: ตัวแปร ชนิดข้อมูล และนิพจน์คณิตศาสตร์:** การเลือกใช้ชนิดข้อมูลที่เหมาะสมตามการใช้หน่วยความจำ
*   **บทที่ 3: การรับและแสดงผลข้อมูลอย่างมีประสิทธิภาพ:** เทคนิคการจัดรูปแบบเอาต์พุตและการรับอินพุตขั้นสูง
*   **บทที่ 4: โครงสร้างควบคุมการตัดสินใจแบบซับซ้อน:** การใช้ Nested If และ Switch-Case ในสถานการณ์จริง
*   **บทที่ 5: การทำซ้ำและลูป:** ความแตกต่างระหว่าง For, While, Do-While และการควบคุมลูป (Break/Continue)
*   **บทที่ 6: ฟังก์ชันและการประมวลผลแบบโมดูล:** การแยกโค้ดเป็นส่วนย่อยเพื่อความสามารถในการนำกลับมาใช้ใหม่
*   **บทที่ 7: การส่งผ่านค่าในฟังก์ชันและการจัดการ Scope:** การใช้งาน Call by Value vs Reference อย่างถูกต้อง
*   **บทที่ 8: อาเรย์และการจัดการข้อมูลชุด:** การจัดเก็บและเข้าถึงข้อมูลในรูปแบบอาเรย์ 1 มิติ และ 2 มิติ
*   **บทที่ 9: การจัดการข้อความและสตริง:** การใช้ไลบรารีสตริงและการค้นหารูปแบบข้อความเบื้องต้น
*   **บทที่ 10: พื้นฐานการอ่านและเขียนไฟล์:** การบันทึกข้อมูลและดึงข้อมูลจากไฟล์ข้อความอย่างง่าย

### 3. 09-131-103 Discrete Structures (9 บท)
*   **บทที่ 1: ตรรกศาสตร์และการพิสูจน์:** การใช้ Truth Tables และกฎทางตรรกศาสตร์
*   **บทที่ 2: ตัวบ่งปริมาณและการประยุกต์ใช้:** การเขียนกฎคอมพิวเตอร์ด้วย Quantifiers
*   **บทที่ 3: ทฤษฎีเซตและการดำเนินการ:** ความสัมพันธ์ระหว่างเซตและการพิสูจน์คุณสมบัติ
*   **บทที่ 4: ความสัมพันธ์และคุณสมบัติทางลอจิก:** การใช้ Relation ในการแทนข้อมูล
*   **บทที่ 5: ฟังก์ชันเชิงคณิตศาสตร์สำหรับคอมพิวเตอร์:** การตรวจสอบ Injective, Surjective และหัวข้อเบื้องต้นของ Recursive Functions
*   **บทที่ 6: ทฤษฎีกราฟพื้นฐานและการประยุกต์:** กราฟแบบระบุทิศทางและไม่ระบุทิศทางในโครงข่าย
*   **บทที่ 7: โครงสร้างต้นไม้และการท่องไป:** วิถีการนำทางในต้นไม้และการประยุกต์ใช้ในอัลกอริทึม
*   **บทที่ 8: หลักการนับและความน่าจะเป็นแบบไม่ต่อเนื่อง:** เทคนิคการนับเบื้องต้น
*   **บทที่ 9: ความสัมพันธ์เวียนเกิดและการอุปนัยเชิงคณิตศาสตร์:** การพิสูจน์ความถูกต้องของโปรแกรมวนซ้ำ

### 4. 09-132-101 Functional Programming (8 บท)
*   **บทที่ 1: แนวคิด Paradigm เชิงฟังก์ชัน:** การเปลี่ยนผ่านจากคำสั่ง (Imperative) สู่ฟังก์ชัน
*   **บทที่ 2: ฟังก์ชันในฐานะพลเมืองชั้นหนึ่ง:** Higher-Order Functions และ Lambda Calculus
*   **บทที่ 3: ความไม่เปลี่ยนแปลงและผลข้างเคียง (Side Effects):** การเขียนโค้ดที่คาดเดาผลลัพธ์ได้ 100%
*   **บทที่ 4: การประมวลผลรายการข้อมูล:** การใช้ Map, Filter, Reduce ในมิติต่างๆ
*   **บทที่ 5: การเรียกซ้ำและการพิสูจน์ความถูกต้อง:** Tail-call Optimization และ Recurrence
*   **บทที่ 6: การจับคู่รูปแบบและชนิดข้อมูล:** ADTs และการดึงข้อมูลอย่างทรงพลัง
*   **บทที่ 7: การประมวลผลแบบเชื่องช้า (Lazy Evaluation):** การจัดการกับ Infinite Lists และ Streams
*   **บทที่ 8: พื้นฐาน Functors และ Monads:** การจัดการ Context และ Error Handling

### 5. 09-132-102 Object Oriented Programming (10 บท)
*   **บทที่ 1: พื้นฐานวัตถุและคลาส:** การนิยาม State และ Behavior
*   **บทที่ 2: การห่อหุ้มและการสิทธิ์เข้าถึง:** การใช้ Private, Public และการทำ Getters/Setters
*   **บทที่ 3: คอนสตรัคเตอร์และการจัดการทรัพยากร:** Lifecycle ของวัตถุในหน่วยความจำ
*   **บทที่ 4: การสืบทอด (Inheritance):** การขยายความสามารถและ Reusability
*   **บทที่ 5: การพหุสัณฐาน (Polymorphism):** Overloading และ Overriding
*   **บทที่ 6: อินเทอร์เฟซและคลาสบทคัดย่อ:** การทำสัญญาระหว่างคลาส (Contract Design)
*   **บทที่ 7: ความสัมพันธ์ของวัตถุเชิงลึก:** Association, Composition, Aggregation
*   **บทที่ 8: การจัดการข้อผิดพลาดระดับวัตถุ:** Exception Hierarchy และการโยนข้อผิดพลาด
*   **บทที่ 9: พื้นฐาน Design Patterns:** Singleton และ Factory เบื้องต้น
*   **บทที่ 10: หลักการ SOLID:** กุญแจสู่การเขียนโปรแกรมที่ดูแลรักษาง่าย

### 6. 09-132-103 Digital and Boolean Algebra (7 บท)
*   **บทที่ 1: ระบบตัวเลขและพีชคณิตบูลีน:** พื้นฐานลอจิกเกต (Logic Gates)
*   **บทที่ 2: การลดรูปวงจรลอจิก:** การใช้ K-Map และทฤษฎีพื้นฐาน
*   **บทที่ 3: วงจรตรรกะแบบผสม (Combinational):** Adders, Encoders, Mux
*   **บทที่ 4: วงจรตรรกะแบบลำดับ (Sequential):** Flip-Flops และ Latches
*   **บทที่ 5: การออกแบบรีจิสเตอร์และตัวนับ:** Counters และ Shift Registers
*   **บทที่ 6: สถานะจำกัด (FSM):** การออกแบบเครื่องจักรสถานะสำหรับคอมพิวเตอร์
*   **บทที่ 7: หน่วยความจำและ ADC/DAC:** การแปลงสัญญาณอนาล็อกสู่ดิจิทัล

## 🔵 ชั้นปีที่ 2: ทักษะวิชาชีพหลัก (Major Core)

### 7. 09-132-201 Database Systems (9 บท)
*   **บทที่ 1: สถาปัตยกรรมและโมเดลฐานข้อมูล:** บทบาทของ DBMS ในองค์กร
*   **บทที่ 2: การออกแบบด้วย ER-Model:** การดึง Requirement เป็นความสัมพันธ์ทางข้อมูล
*   **บทที่ 3: โมเดลความสัมพันธ์ (Relational Model):** ทฤษฎีเซตเบื้องหลังฐานข้อมูล
*   **บทที่ 4: ภาษา SQL สำหรับการจัดการข้อมูล:** DDL, DML, DQL และการ Join ขั้นสูง
*   **บทที่ 5: การทำ Normalization:** การลดความซ้ำซ้อนสู่ความถูกต้อง (1NF-BCNF)
*   **บทที่ 6: การจัดการ Transaction:** ACID Properties และการควบคุมสภาวะพร้อมกัน
*   **บทที่ 7: การบริหารความปลอดภัย:** การกำหนดสิทธิ์ผู้ใช้และการสำรองข้อมูล
*   **บทที่ 8: การจัดการดัชนี (Indexing):** การเพิ่มความเร็วในการสืบค้นด้วย B-Tree
*   **บทที่ 9: แนวคิดฐานข้อมูล NoSQL:** บทนำสู่ฐานข้อมูลยุคใหม่ที่ไม่ใช้โครงสร้างตาราง

### 8. 09-131-201 Numerical Methods (8 บท)
*   **บทที่ 1: ความคลาดเคลื่อนในคำนวณคอมพิวเตอร์:** แหล่งที่มาของ Error
*   **บทที่ 2: การหารากของสมการ:** วิธี Bisection และ Newton-Raphson
*   **บทที่ 3: ระบบสมการเชิงเส้น:** การใช้ Gaussian Elimination
*   **บทที่ 4: การประมาณค่าในช่วง (Interpolation):** การเชื่อมโยงจุดข้อมูล
*   **บทที่ 5: การพยากรณ์ด้วย Curve Fitting:** การหาเส้นวิเคราะห์แนวโน้ม
*   **บทที่ 6: การหาอนุพันธ์เชิงตัวเลข:** การคำนวณอัตราเปลี่ยนแปลง
*   **บทที่ 7: การอินทิเกรตเชิงตัวเลข:** วิธี Trapezoidal และ Simpson’s rule
*   **บทที่ 8: การแก้สมการเชิงอนุพันธ์:** วิธีการเชิงตัวเลขขั้นสูง

### 9. 09-132-202 Data Structures (10 บท)
*   **บทที่ 1: Linked Lists:** การจัดการหน่วยความจำแบบพลวัต
*   **บทที่ 2: Stacks & Queues:** การประยุกต์ใช้ในซอฟต์แวร์ระบบ
*   **บทที่ 3: Binary Trees:** การออกแบบและท่องไปในต้นไม้
*   **บทที่ 4: Balanced Trees:** AVL และ Red-Black Trees เบื้องต้น
*   **บทที่ 5: Binary Heaps:** สำหรับ Priority Queues
*   **บทที่ 6: Hashing:** การสืบค้นระดับ O(1) และการแก้ปัญหา Collision
*   **บทที่ 7: Graphs:** การแทนค่าและอัลกอริทึม BFS, DFS
*   **บทที่ 8: Advanced Sorting:** Merge และ Quick Sort
*   **บทที่ 9: External Storage:** โครงสร้าง B-Tree
*   **บทที่ 10: การคัดเลือกโครงสร้างข้อมูล:** การตัดสินใจเชิงวิศวกรรม

### 10. 09-132-203 Computer Org/Arch (9 บท)
*   **บทที่ 1: สถาปัตยกรรม CPU:** ชุดคำสั่งและการจัดการ Register
*   **บทที่ 2: ALU:** การคำนวณตัวเลขและลอจิกภายใน
*   **บทที่ 3: หน่วยควบคุม (Control Unit):** กลไกการบริหารจังหวะ
*   **บทที่ 4: Pipelining:** การประมวลผลควบคู่
*   **บทที่ 5: Memory Hierarchy:** แคช L1-L3
*   **บทที่ 6: Virtual Memory:** การบริหารหน่วยความจำเสมือน
*   **บทที่ 7: I/O Bus:** การสื่อสารรับส่งข้อมูลภายใน
*   **บทที่ 8: Parallel Computing:** Multi-core ระบบ
*   **บทที่ 9: ทิศทางสถาปัตยกรรมอนาคต:** แนวคิดชิปประมวลผลยุคใหม่

### 11. 09-132-204 Operating Systems (9 บท)
*   **บทที่ 1: โครงสร้าง OS:** Kernel และ System Calls
*   **บทที่ 2: การจัดการโพรเซส:** Lifecycle และ Concurrency
*   **บทที่ 3: CPU Scheduling:** อัลกอริทึมการจัดคิวงาน
*   **บทที่ 4: Semaphores:** การป้องกันการเข้าถึงพร้อมกัน
*   **บทที่ 5: Deadlock Management:** การป้องกันและการตรวจจับ
*   **บทที่ 6: Memory Management:** การจัดสรรและป้องกัน
*   **บทที่ 7: Paging:** ระบบหน่วยความจำแบบหน้า
*   **บทที่ 8: File Systems:** การบริหารจัดการระบบดิสก์
*   **บทที่ 9: OS Security:** การป้องกันมัลแวร์ระดับระบบ

### 12. 09-132-205 Human-Computer Interaction (7 บท)
*   **บทที่ 1: จิตวิทยาผู้ใช้:** สายตา ความจำ และการเรียนรู้
*   **บทที่ 2: โมเดลการโต้ตอบ:** GUI และ Touch Interfaces
*   **บทที่ 3: กระบวนการออกแบบ UCD:** การออกแบบเน้นมนุษย์
*   **บทที่ 4: Prototyping:** จากกระดาษสู่ดิจิทัล
*   **บทที่ 5: Usability Evaluation:** การทดสอบความยากง่าย
*   **บทที่ 6: Accessibility:** การออกแบบสำหรับความเท่าเทียม
*   **บทที่ 7: Modern HCI:** อุปกรณ์สวมใส่และ IoT

### 13. 09-132-206 Artificial Intelligence (10 บท)
*   **บทที่ 1: ปรัชญาและประเภท AI:** ประวัติศาสตร์และความหมาย
*   **บทที่ 2: ตัวแทนอัจฉริยะ (Agents):** โครงสร้างและการตัดสินใจ
*   **บทที่ 3: การค้นหาเชิงลึก (Search):** อัลกอริทึมพื้นฐานสู่ A*
*   **บทที่ 4: Heuristics:** การค้นหาแบบอิงความรู้เบื้องต้น
*   **บทที่ 5: ฐานความรู้และลอจิก:** การแทนค่าสู่กฎตรรกะ
*   **บทที่ 6: Machine Learning:** บทนำสู่คลังข้อมูลการเรียนรู้
*   **บทที่ 7: Neural Networks:** พื้นฐานระบบประสาทเทียม
